# Rolevate Hero Section

A Next.js 14 project recreating the Google Antigravity hero section with React Three Fiber.

## Features

- Particle system with polar orbit motion
- Mouse velocity-based field warp
- Layered breathing oscillation
- Global field rotation
- Radial distribution with center clustering

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) in your browser.

## Tech Stack

- Next.js 14.2.18
- React 18.2.0
- React Three Fiber
- Three.js
- Tailwind CSS
